# 🎓 Review & Rating System

A modern and responsive review platform built to allow users to share experiences, submit ratings, and explore detailed feedback through a clean and professional interface.

The platform includes advanced review analytics, admin moderation controls, dynamic filtering, and a fully responsive dark-themed UI designed for a smooth user experience.

---

# ✨ Features

## 👨‍💻 Review System
- Submit detailed reviews
- Star-based rating system
- Category-wise ratings
- Verified reviewer interface
- Helpful reaction support

## ⭐ Rating Analytics
- Overall rating calculation
- Dynamic rating distribution
- Real-time review statistics
- Category performance insights

## 🛡️ Admin Dashboard
- Review moderation system
- Approve / Reject reviews
- Search and filter functionality
- Review management table

## 🔍 Smart Filtering & Sorting
- Search reviews instantly
- Sort by newest and oldest
- Filter by highest and lowest ratings
- Dynamic review updates

## 📱 Responsive UI
- Mobile-friendly layout
- Modern dark-themed design
- Optimized user experience
- Smooth responsive interface

# ⚙️ Environment Variables

Create a `.env` file in the root directory and add:

```env
REACT_APP_SUPABASE_URL=your_supabase_project_url
REACT_APP_SUPABASE_ANON_KEY=your_supabase_anon_key
```

# 🚀 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/chandana31zz/review-rating-system.git
```

## Navigate to Project Folder

```bash
cd review-rating-system
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm start
```

Application runs on:

```bash
http://localhost:3000
```
